Metrics
=======

Retrieve a list of campaigns
----------------------------

.. code-block:: python

    from sparkpost import SparkPost

    sp = SparkPost()

    sp.metrics.campaigns.list()


Retrieve a list of domains
--------------------------

.. code-block:: python

    from sparkpost import SparkPost

    sp = SparkPost()

    sp.metrics.domains.list()
